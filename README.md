## 🚀 Featured Project: [SafePath]
> **"우리는 단순한 지도 앱이 아닙니다. 학부모를 위한 '안전 컨설턴트'입니다."**

**1. 🎯 문제 정의 (The Gap)**
- **Plan A의 실패:** 맞벌이 등 현실적인 이유로 자녀와 24시간 동행할 수 없는 부모들의 불안감.
- **기존 서비스의 한계:** 네이버지도/카카오는 '최단 거리(효율)'만 강조하여 '안전'에 대한 니즈를 무시함.

**2. 💡 솔루션 (Safety Report)**
- **Safe Scoring Engine:** 공공 데이터(CCTV, 경찰서, 가로등, 24시 편의점)를 분석하여 경로별 안전 점수 산출.
- **AI Consulting:** Upstage AI(LLM)를 활용해 "B경로는 2분 더 걸리지만 CCTV가 7개 더 많아 안전합니다"라는 전문 리포트 제공.
